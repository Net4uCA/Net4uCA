Natural Language Processing

From Fundamentals to Large Language Models

10 Hours

Course Overview

A comprehensive introduction to Natural Language Processing, covering both traditional approaches and modern Large Language Models. The course combines theoretical foundations with hands-on workshops.

What You'll Learn

  • Text Classification Fundamentals
  • Modern NLP Techniques
  • Large Language Models
  • RAG Implementation

Prerequisites

  • Basic Python programming
  • Basic ML concepts

Course Syllabus

1. Fundamentals of Text Classification (2h)

  • Course overview and objectives
  • Real-world text classification applications
  • Tokenization, stop words, stemming
  • Workshop: Building a text classifier

2. Modern Text Classification (2h)

  • From one-hot encoding to dense representations
  • BERT architecture
  • Contextual embeddings
  • Workshop: BERT hands-on

3. Large Language Models (2h)

  • LLM Theory and Knowledge Acquisition
  • Training and Instruction Tuning
  • Strengths and Limitations
  • Hands-on: Zero-shot and Few-shot Learning

4. RAG Implementation (3h)

  • RAG Fundamentals and Architecture
  • Vector Databases
  • Practical Implementation Workshop
  • Adding Search Capabilities

5. Future Trends and Assessment (1h)

  • Emerging Trends and Research Directions
  • Industry Applications
  • Mini-Project Development
  • Optional Final Assessment (15 min)
