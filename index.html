Natural Language Processing

From Fundamentals to Large Language Models

20 Hours

Detailed Course Syllabus

1. Introduction to Text Classification (1.5h)

  • Course overview and objectives
  • Brief history: from rule-based to modern approaches
  • Key applications and Classification tasks:
    • Sentiment analysis
    • Topic classification
    • Language identification
    • Spam detection

2. Fundamentals of Text Processing (1.5h)

  • Text preprocessing fundamentals:
    • Tokenization
    • Stop word removal
    • Stemming and lemmatization
  • Representations:
    • Bag of Words (BOW)
    • TF-IDF vectorization
  • Model evaluation metrics:
    • Accuracy, Precision, Recall, F1-score
    • Confusion matrix

3. Hands-on (1h)

  • Building a simple spam detector using BOW/TF-IDF
  • Basic evaluation and interpretation of results

4. Word Embeddings (1.5h)

  • From one-hot encoding to dense representations
  • Word2Vec, GloVe, FastText
  • Properties and limitations of word embeddings
  • Visualizing word embeddings

5. Modern Text Classification (1.5h)

  • Contextual embeddings
  • Introduction to BERT and transformers

6. Hands-on (1h)

  • Using pre-trained BERT for classification
  • Comparing different approaches

7. Large Language Models (1.5h)

  • Architecture and capabilities
  • Key players: GPT, Gemini, Claude, LLama
  • Strengths and limitations
  • Ethical considerations

8. Advanced Evaluation Metrics (1.5h)

  • NLG evaluation metrics: BLEU, ROUGE, METEOR
  • BERTScore
  • Human evaluation
  • Cross-lingual evaluation

9. Interactive Demo Session (1h)

  • Exploring LLM capabilities with Gemini
  • Prompt engineering practice
  • Real-world applications

10. Working with LLMs (1h)

  • Fine-tuning strategies
  • Parameter-efficient fine-tuning (PEFT)
  • Best practices and common pitfalls

11. Advanced Applications (30 Minutes)

  • Zero-shot and few-shot learning
  • Chain-of-thought prompting

12. Hands-on (2.5h)

  • Practical fine-tuning exercises
  • Quantization
  • Performance evaluation

    • 13. Retrieval-Augmented Generation (3h)

    • RAG architecture and components
    • Vector databases
    • Integration with LLMs
    • Advanced retrieval strategies
    • Hands-on Laboratory

    14. Future Trends (30min)

    • Latest developments in the field
    • Open challenges and research opportunities
    • Industry applications and future directions

    15. Final Assessment (30 Min, Optional)

    • Final Exam/ Project
