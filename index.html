University of Cagliari

Seminar on Text Classification & Natural Language Processing

From Text Classification to Large Language Models

Duration: 10 Hours (5 Lessons) February 2025

Course Overview

A comprehensive seminar covering modern text classification techniques, from fundamentals to advanced LLM applications. The course combines theoretical foundations with hands-on practice, emphasizing real-world applications.

Course Materials Repository

Learning Objectives

  • Understand fundamental concepts of text classification
  • Master modern NLP techniques and architectures
  • Gain practical experience with LLMs
  • Develop hands-on skills in RAG implementation

Course Structure

  • 5 intensive lessons (2 hours each)
  • Theoretical foundations + hands-on workshops
  • Practical exercises and real-world applications

Course Syllabus

Lesson 1: Text Classification and Natural Language Processing

February 3rd - 2 hours

  • Introduction to NLP and its importance
  • Applications: Sentiment Analysis, Machine Translation, Speech Recognition
  • Text Classification Tasks and Real-world Applications
  • NLP Pipeline: Data Acquisition, Preprocessing, Feature Engineering
  • Workshop: Building a basic sentiment classifier

Lesson 2: Modern Text Classification

February 4th - 2 hours

  • Tokens and Embeddings Fundamentals
  • Word Embeddings: Word2Vec, BERT
  • Beyond BERT: Modern Architectures
  • Workshop: Tokenizers and Embeddings in Practice

Lesson 3: Generative Large Language Models

February 5th - 2 hours

  • From Discriminative to Generative Models
  • Understanding Model Parameters and Resources
  • LLM Ecosystem: Closed vs Open Models
  • Context Windows and Prompting Techniques

Lesson 4: Retrieval Augmented Generation

February 6th - 2 hours

  • LLM Limitations and RAG Solutions
  • Semantic Search and Dense Retrieval
  • Document Chunking Strategies
  • RAG Implementation Workshop

Lesson 5: Complex Applications with Language Models

February 6th - 2 hours

  • Web Scraping Agents
  • Data Augmentation Techniques
  • Hallucination Detection
  • Specialized Agents Development
